Release Notes for Translate AS1 for C++, Version 1.0.6.

Bug Fixes
Floating point numbers now supported.
New Features
Conditional compilation directives now supported.

Bug Fixes

1. Floating point numbers now supported.

The grammar now parses floating-point numbers correctly.

var i = 0.0;
var j = .0;
var k = 1.0e10;

In previous releases of Translate AS1 only integer numbers were parsed correctly.

New Features

1. Conditional compilation directives now supported.

The preprocessor now implements #define, #ifdef, #ifndef, #else and #endif directives:

#define DIRECTIVE 1

#ifdef DIRECTIVE
    a = 1;
    b = 2;
#else
    a = 3;
    b = 4;
#endif

#ifndef DIRECTIVE
    c = 1;
    d = 2;
#else
    c = 3;
    d = 4;
#endif

Is equivalent to the script:

    a = 1;
    b = 2;

    c = 3;
    d = 4;
#endif

There are a few limitations to the current implementation of the directives:

  • The values associated with #define directives can only use interger literals.
  • Directives cannot be nested.

The functionality supported in directives will be expanded with later releases.

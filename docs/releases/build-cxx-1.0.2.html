Release Notes for Translate AS1 for C++, Build 3.

Bug Fixes
instanceof operator is now supported.
delete operator is now supported.

Bug Fixes

1. instanceof operator is now supported.

In Flash 6 (Flash MX) the instanceof operator was added to Actionscript to allow runtime type checking of objects:

    var obj = new MyClass;

    if (obj instanceof MyClass)
    {
        ...
    }

This build updates the parser to support the instanceof operator. Previous releaes would generate an error when parsing a script containing instanceof.

2. delete operator is now supported.

In previous releases of Translate objects or variables were deleted using the delete function:

    var obj = new MyClass;

    delete(obj);

This release now also allows delete to be used as an unary operator:

    var obj = new MyClass;

    delete obj;

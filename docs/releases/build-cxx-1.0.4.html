Release Notes for Translate AS1 for C++, Build 5.

Bug Fixes
Multiple assignments now support object attributes.

Bug Fixes

1. Multiple assignments now support object attributes.

In ActionScript several variables can be assigned a value in a single statement:

    var a = b = c = 0;

The previous release of Translate added support for multiple assignments, however only variables were supported. Code that included dereferencing an object's attributes caused a parse error to be generated:

    var a = object.attribute = 1;

This release updates the parser to support both variables and attributes in assignment statements.
